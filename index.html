Tween 82.8 って何？

現実と夢の狭間に流れるFMラジオをコンセプトにした、ジャンル別BGMプレイヤー。

名前の由来：Tween = between（狭間）／ 82.8 = 太陽系が銀河を公転する速度（82.8万 km/h）をFM周波数に見立てた

  • 44ジャンルに対応（Lofi, Jazz, City Pop, Synthwave...）
  • YouTubeの1時間ミックスを即再生
  • Windows 95風のレトロUI
  • 広告なし・無料・PWAアプリ対応（ホーム画面に追加可能）
  • 勉強・作業・眠れない夜のお供に

きっかけ

ちょっと聞いていいですか — 深夜ラジオ、聴いたことある人？

1990年代後半。バブルが弾けて、大人たちは「もうダメだ」と言ってた。
青春のまっただ中にいた人たちは、それを聞きながら育った。将来への漠然とした不安。
でも、深夜だけは自由だった。
深夜ラジオ、深夜アニメ、深夜のネット。
大人が寝た後の時間だけが、自分だけの世界だった。

仕事をしながら、通勤しながら、眠れない夜。
ふと、あの頃の空気が欲しくなる瞬間がある。
曲を選ぶんじゃなくて、ただ耳を傾けるだけの場所。
FMラジオのように流しっぱなしにできる、あの感覚。

そもそもレトロが好きだった。
だから作った。選ばなくていい音楽と、あの時代の空気を、もう一度。

Tween 82.8 — 「場所」を作る

Spotify、Apple Music、YouTube Music — 音楽を聴くサービスはいくらでもある。
でも欲しかったのはサービスじゃない。音楽を聴く「場所」が欲しかった。
BGMプレイヤーを作ったんじゃない。あの空気を再現するための装置を作った。

Tween = between。現実と夢の狭間。

存在しない周波数。どこでも受信できないラジオ局。

45ジャンル。Lofi、Jazz、City Pop、Synthwave...
チューニングだけが、あなたの仕事。世界観があるから「ここで聴きたい」が生まれる。

なぜ Windows 95？

C:\> 深夜にFMラジオを聴いていた人は、
C:\> あの時代、CRTモニターの前に座っていた人と同じ。
C:\> 画面の光と、スピーカーからの音だけが世界だった夜。

選ばなかった理由
× 懐かしいから
× 流行りだから
× 見た目がかわいいから

選んだ理由
✓ UIと中身が同じ情緒を指している
✓ 「場所」の空気を成立させる唯一のUI
✓ 体験の設計として必然

レトロな見た目を被せたんじゃない。深夜ラジオの空気を画面にしたらWin95になった。
スキューモーフィズム — 画面の向こうに「部屋」があると感じさせるため。貼り皮じゃなく、必然。

Win95 デザイン原則

Win95のデザインには思想がある。見た目のルールじゃなくて、「画面の向こうに物がある」と感じさせる設計。

思想

物理の模倣 — ボタンは出っ張り、押したら凹む。光と影だけで「触れる」感覚を作る
制約が美学 — 色5色、フォント1つ、グラデーションなし。制約があるから統一感が生まれる
空間の階層 — デスクトップ > ウィンドウ > ダイアログ。z軸で「部屋の奥行き」を表現

実装

光源は左上 — 左上が白、右下がグレー。2色で立体感
状態は反転 — 押したらボーダーが反転。hover変色じゃなく凹む
色は5色 — #c0c0c0 / #fff / #808080 / #404040 / #000080

AIとの付き合い方

「AIは人の差をなくす」とよく言われる。
確かに入り口は低くなった。でも、使い方の差が、成果の差をさらに大きくする。

実例：この資料を作った時の話
AIに「きっかけスライドを書いて」と頼んだら、
「1995年8月24日、深夜0時。世界中の人が家電量販店の前に並んだ」と書いてきた。
綺麗だけど、自分の言葉じゃない。嘘くさい。
「BGMプレイヤーじゃなくて、あの空気を再現する装置だ」と伝えたら、ようやく芯に近づいた。
AIは方向を出す。でも「これは違う」「こっちだ」と判断するのは人間の仕事。

AIにできること人間がやること
大量生成、パターン認識、高速化「これは違う」と言う。方向修正は人間だけ
文脈を渡せば精度が上がるCLAUDE.md / skills で知識を仕込む
素材を出す、選択肢を並べる選ぶ・捨てる・書き直す
